# Devuan Security OS

Кастомний дистрибутив на базі **Devuan Excalibur 6.x** (без systemd), орієнтований на безпеку та мінімалізм.

## Про проєкт

- **База:** Devuan Excalibur 6.1.1 (на основі Debian 13 Trixie)
- **Init система:** SysVinit (без systemd)
- **Середовище:** XFCE з кастомним налаштуванням у стилі Windows
- **Інструмент збірки:** live-build
- **Архітектура:** amd64
